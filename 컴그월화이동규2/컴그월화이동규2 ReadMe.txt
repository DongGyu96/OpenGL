실행방법-
일반적인 방법으로 실행

명령어-
 1: 맑은 날씨
 2: 눈오는 날씨
 3: 비오는 날씨

 +:(선로 건설 모드) 다음에 지을 선로의 높이 증가
   (화면 뷰 모드) z축으로 화면 이동
 -:(선로 건설 모드) 다음에 지을 선로의 높이 감소
   (화면 뷰 모드) z축으로 화면 이동

 w,a,s,d : (화면 뷰 모드) x축/y축 각각 회전

 c: 선로 건설이 끝난 상황에서 카메라 모드 변경
    화면 뷰->3인칭 코스터 뷰->1인칭 코스터 뷰->화면 뷰

 m: (선로 건설 모드) 직선 생성 모드 On/Off

 스페이스바 : (선로 건설 모드가 아닐 경우)
              코스터에서 총알 발사
 q: 종료

구현한 내용-
총 33번 클릭으로 제어점 생성
(스테이션 3개 이후 각각 2번의 클릭으로 선로 생성)
2차 베지어 곡선을 통한 선로 건설
직선 모드를 켜서 곡선이 아닌 직선이 생성되며
가장 최근에 생긴 직선 구간에 터널 생성
처음 건설하는 스테이션은 무조건 직선으로 생성
날씨 구현, 총알 발사 구현, 1인칭, 3인칭 화면 구현
선로의 높낮이, 직선모드 변경이 있을경우 콘솔창에 출력
선로 건설시 나무에 클릭할 경우 건설 불가능
롤러코스터가 선로에 맞춰서 회전후 이동
롤러코스터가 선로 끝에 도달할경우 처음 선로로 이동

구현 못한 내용-
오르막, 내리막에 따른 속도변화 미구현

추가구현한 내용-
롤러코스터를 쫒아가는 3인칭 시점 구현
